# Stride Fitness Tracker 🏃‍♂️💨

**Stride** is a premium, cross-platform fitness tracking application built with Flutter and Firebase. It offers a sleek, modern experience for users to track their activities, visualize their progress, and stay motivated on their fitness journey.

![Stride Logo](https://img.shields.io/badge/Status-Complete-success?style=for-the-badge&logo=flutter)
![Firebase](https://img.shields.io/badge/Backend-Firebase-orange?style=for-the-badge&logo=firebase)

---

## ✨ Key Features

### 🔐 Advanced Authentication
- **Multi-platform Login:** Support for Google, GitHub, and Email/Password.
- **Guest Mode:** Try the app instantly without an account.
- **Security:** Fully integrated with Firebase Auth for secure session management.

### 📊 Dynamic Dashboard
- **Real-time Stats:** Track steps, calories, and active minutes.
- **Weekly Progress:** Beautiful, animated bar charts showing your activity trends.
- **Motivational Quotes:** Get a fresh boost of inspiration every time you open the app.

### 🏃‍♂️ Smart Tracking
- **Auto-counting:** Automatically tracks steps on mobile devices using physical sensors.
- **Manual Logging:** Log workouts like Running, Cycling, or Yoga manually with ease.
- **Step Goals:** Set personal daily goals and track your completion percentage.

### 🎨 Premium UI/UX
- **Modern Aesthetics:** Liquid splash screens, glassmorphism cards, and smooth transitions.
- **Dark Mode:** Full support for high-contrast Dark Mode to save battery and reduce eye strain.
- **Responsive:** Works perfectly on Web, Android, and iOS.

---

## 🛠 Tech Stack

- **Frontend:** [Flutter](https://flutter.dev) (Dart)
- **State Management:** [Provider](https://pub.dev/packages/provider)
- **Database:** [Google Cloud Firestore](https://firebase.google.com/docs/firestore)
- **Auth:** [Firebase Authentication](https://firebase.google.com/docs/auth)
- **Charts:** [Fl Chart](https://pub.dev/packages/fl_chart)

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Firebase project configured

### Installation
1. **Clone the repo:**
   ```bash
   git clone https://github.com/CODEX001-TECH/CodeAlpha_FitnessTracker.git
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the app:**
   ```bash
   flutter run -d chrome  # For Web
   ```
